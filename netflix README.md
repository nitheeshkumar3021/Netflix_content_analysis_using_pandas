# Netflix Data Analysis using Pandas

## Project Overview
This project analyzes the Netflix dataset using Python and Pandas to understand trends in movies and TV shows available on Netflix. The analysis includes data cleaning, exploration, and visualization to extract useful insights.

## Objective
The main objective of this project is to explore the Netflix dataset and answer important questions such as:
- How many Movies and TV Shows are available on Netflix?
- Which country produces the most Netflix content?
- What are the most common genres on Netflix?
- How has Netflix content grown over the years?

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset Information
The dataset contains the following columns:
- show_id
- type
- title
- director
- cast
- country
- date_added
- release_year
- rating
- duration
- listed_in
- description

## Data Analysis Steps
1. Data Loading using Pandas
2. Data Cleaning and Handling Missing Values
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Extracting Insights from the Dataset

## Key Insights
- Netflix contains a large number of Movies compared to TV Shows.
- The United States produces the majority of Netflix content.
- Drama and International Movies are among the most popular genres.
